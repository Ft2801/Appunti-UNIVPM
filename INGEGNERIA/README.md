# Appunti Ingegneria - UNIVPM

Repository di appunti, slide, esercizi e materiale didattico per i corsi dell'Università Politecnica delle Marche (UNIVPM).

## Utilizzo

Naviga nella cartella del tuo corso di laurea e poi nella materia di interesse.
Ogni cartella contiene un README con la descrizione del corso e il materiale caricato dai contributori.

## Formati supportati

Sono accettati file nei seguenti formati: PDF, DOCX, PPTX, XLSX, TXT, MD, PNG, JPG, SVG.
