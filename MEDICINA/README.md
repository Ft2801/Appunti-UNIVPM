# Medicina e Chirurgia

Appunti, slide, esercizi e materiale didattico per i corsi della Facoltà di Medicina e Chirurgia dell'Università Politecnica delle Marche (UNIVPM).

## Struttura

Questo repository è organizzato nei seguenti percorsi di studio:

- **ciclo-unico**: Corsi di laurea a ciclo unico (6 anni) - Medicina e Chirurgia, Odontoiatria
- **triennale**: Corsi della laurea triennale e profili sanitari (Infermieristica, Fisioterapia, ecc.)
- **magistrale**: Corsi della laurea magistrale e specializzazioni sanitarie

## Utilizzo

1. Naviga nel percorso di studio che ti interessa (ciclo-unico, triennale o magistrale)
2. Seleziona il corso desiderato
3. Ogni cartella contiene un README con la descrizione del corso e il materiale caricato dai contributori

## Formati supportati

Sono accettati file nei seguenti formati: PDF, DOCX, PPTX, XLSX, TXT, MD, PNG, JPG, SVG.
