# Linee guida per i contributori

## Come aggiungere materiale

1. Fai un fork della repository.
2. Crea un branch con un nome descrittivo, ad esempio: aggiunta-appunti-analisi-1.
3. Aggiungi il materiale nella cartella corretta (corso > materia).
4. Apri una pull request descrivendo brevemente il contenuto aggiunto.

## Regole sul materiale

- Carica solo materiale di tua produzione oppure con licenza libera.
- Non caricare testi d'esame protetti da copyright senza autorizzazione del docente.
- Non caricare soluzioni ufficiali distribuite dai docenti senza consenso.
- I file devono essere leggibili: preferisci PDF rispetto a formati proprietari quando possibile.

## Nomenclatura dei file

Usa nomi chiari e descrittivi, senza spazi (usa trattini o underscore). Esempi:

    appunti-lezione-01.pdf
    esercizi-cap3-con-soluzioni.pdf
    slide-docente-settimana-04.pptx
    formula-sheet-esame.pdf
    esame-giugno-2023.pdf

## Struttura delle cartelle

Non creare sottocartelle aggiuntive senza discuterne prima in una issue.
La struttura attuale rispecchia i piani di studio ufficiali UNIVPM 2025/26.
