# Appunti UNIVPM

Repository di appunti, slide, esercizi e materiale didattico per i corsi dell'Università Politecnica delle Marche (UNIVPM).

## Facoltà

Questa repository contiene il materiale didattico per le seguenti facoltà:

- **[INGEGNERIA](INGEGNERIA/)** - Corsi di Ingegneria organizzati per percorso (triennale/magistrale)
- **[ECONOMIA](ECONOMIA/)** - Corsi di Economia organizzati per percorso (triennale/magistrale)
- **[MEDICINA](MEDICINA/)** - Corsi di Medicina e profili sanitari organizzati per percorso (triennale/magistrale)
- **[SCIENZE](SCIENZE/)** - Corsi di Scienze organizzati per percorso (triennale/magistrale)
- **[SCIENZE-AGRARIE](SCIENZE-AGRARIE/)** - Corsi di Scienze Agrarie organizzati per percorso (triennale/magistrale)

## Utilizzo

1. Seleziona la facoltà di interesse tra quelle elencate sopra
2. Naviga nel percorso di studio (triennale o magistrale)
3. Scegli il corso desiderato
4. Ogni cartella contiene un README con la descrizione del corso e il materiale caricato dai contributori

## Formati supportati

Sono accettati file nei seguenti formati: PDF, DOCX, PPTX, XLSX, TXT, MD, PNG, JPG, SVG.
